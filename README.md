## Instalação

Após a instalção das ferramentas acima, basta rodar os comandos em seu terminal

* Atualização das dependências do node.js
    * `$ yarn`
* Iniciando serviço
    * `$ yarn start`
* Serviço ficará disponível na host 
    * `http://127.0.0.1:3000`
* Parando serviço
    * `$ yarn stop`
